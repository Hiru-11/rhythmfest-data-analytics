## RhythmFest Data Analytics & Business Intelligence Project

Overview:

This is an academic data analytics and business intelligence project developed as part of a coursework assignment. The project analyzes RhythmFest festival data to generate insights for decision-making using Python and Tableau.

Objective:

To transform raw festival data into meaningful insights that support strategic, tactical, and operational decision-making in event management.

Dataset
1. Raw dataset: RythmFest_Full_Data.xlsx
2. Cleaned dataset: RythmFest_Cleaned_Data.xlsx

Approach
1. Data cleaning and preprocessing using Python
2. Exploratory data analysis to identify patterns and trends
3. Data visualization and dashboard development using Tableau
4. Generation of insights for decision support

Key Analysis Areas
1. Ticket sales and revenue trends
2. Audience attendance and demographic patterns
3. Event performance and operational insights
4. Crowd distribution across festival stages

Tools & Technologies
1. Python (Pandas, NumPy, Matplotlib, Seaborn)
2. Tableau (Data Visualization & Dashboarding)
3. Microsoft Excel (Data storage and preprocessing)

Files Included
1. RythmFest.py → Python analysis script
2. RythmFest.twb → Tableau dashboard file
3. RythmFest_Full_Data.xlsx → Raw dataset
4. RythmFest_Cleaned_Data.xlsx → Processed dataset

Note:

This project was completed as part of an academic coursework assignment focusing on data analytics, business intelligence, and decision support systems.
